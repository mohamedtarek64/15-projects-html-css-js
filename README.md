# 15 Front-End Projects

Created by Mohamed Elkenany.

This collection contains 15 polished HTML, CSS, and JavaScript projects—each living in its own folder with a complete `index.html`, modular styling, and dedicated scripts.

## Projects Included

- `photo-gallery-lightbox` – Dynamic gallery with keyboard-accessible lightbox interactions.
- `interactive-travel-landing` – Destination landing page with programmatic itinerary highlights.
- `fitness-tracker-dashboard` – Workout dashboard visualising weekly stats and goals.
- `recipe-finder-app` – Ingredient search, filters, and responsive recipe cards.
- `weather-insights-widget` – Animated forecasts and climate breakdowns.
- `Qur'an-player-ui` – Qur'an recitation studio powered by the bundled local MP3 track.
- `news-digest-reader` – Curated headlines, filters, and reading list management.
- `crypto-price-tracker` – Real-time style cards, timelines, and portfolio snapshots.
- `event-countdown-timer` – Conference countdown with live schedule overview.
- `language-learning-quiz` – Progressive quiz with speech synthesis feedback.
- `personal-budget-planner` – Envelope budgeting and savings tracking dashboard.
- `movie-review-hub` – Critic aggregates, watchlist, and genre filtering.
- `smart-home-control` – Room automation toggles, energy gauges, and security status.
- `task-manager-kanban` – Kanban board representation with backlog and activity feed.
- `yoga-class-schedule` – Studio timetable, instructor cards, and booking prompts.

Each project is self-contained: open its `index.html` in a browser to explore, or use a static server for best asset loading.
